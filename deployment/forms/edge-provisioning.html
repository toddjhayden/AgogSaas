What is an Edge Computer?

An edge computer runs at your facility (on-premises) and enables offline operations. Even when internet goes down, your workers can create orders, update inventory, and run production. Changes sync to cloud automatically when internet returns.

Typical Use Cases: New facility opening, facility relocation, hardware upgrade, disaster recovery replacement
