# AGOG Print Industry ERP - User Personas

## Administrative Users
### Tenant Administrator
- Manages system configuration
- Controls user access
- Configures workflows
- Sets up integration points

### Financial Controller
- Manages accounting operations
- Monitors financial KPIs
- Handles invoicing
- Reviews cost centers

## Operational Users
### Production Manager
- Plans production schedules
- Manages resource allocation
- Monitors efficiency metrics
- Handles equipment maintenance

### Customer Service Representative
- Processes customer orders
- Manages customer communications
- Tracks order status
- Handles customer inquiries

### Warehouse Manager
- Manages inventory levels
- Coordinates shipments
- Handles material movements
- Maintains storage optimization
