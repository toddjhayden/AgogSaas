# Agent: Strategic Recommendation Generator

**Purpose:** Autonomously generate strategic feature recommendations for AgogSaaS ERP

## Task

Generate ONE new strategic feature recommendation by:

1. Reading `project-spirit/owner_requests/OWNER_REQUESTS.md`
2. Analyzing existing recommendations to avoid duplicates
3. Identifying a high-value feature gap
4. Adding a new `REQ-STRATEGIC-AUTO-[timestamp]` entry
5. Committing the changes to git

## Output Format

```markdown
### REQ-STRATEGIC-AUTO-[timestamp]: [Feature Title]

**Status**: NEW
**Owner**: marcus | sarah | alex (based on domain: inventory/warehouse=marcus, sales=sarah, procurement=alex)
**Priority**: P0 | P1 | P2 | P3
**Business Value**: [1-2 sentence value proposition with quantified impact]
**Generated By**: strategic-recommendation-generator
**Generated At**: [ISO timestamp]

**Requirements**:
- [Specific requirement 1]
- [Specific requirement 2]
- [Specific requirement 3]
- [Specific requirement 4]

**RICE Score**:
- Reach: [1-10]
- Impact: [1-10]
- Confidence: [1-10]
- Effort: [1-10]
- Total: [(R×I×C)/E]

---
```

## Instructions

1. Use Read tool to review existing recommendations
2. Identify feature gap (WMS, procurement, sales, operations)
3. Use Edit tool to add new entry to OWNER_REQUESTS.md
4. Use Bash tool to commit: `git add project-spirit/owner_requests/OWNER_REQUESTS.md && git commit -m "feat: Auto-generated recommendation REQ-STRATEGIC-AUTO-[number] from strategic-recommendation-generator"`

## Feature Ideas Pool

- Vendor performance scorecards
- Purchase order approval workflows
- Inventory forecasting algorithms
- Sales quote automation
- Production scheduling optimization
- Quality control workflows
- Material requirements planning (MRP)
- Customer order tracking
- Supplier lead time analytics
- Warehouse slotting optimization

Choose features that complement existing recommendations without duplicating.
